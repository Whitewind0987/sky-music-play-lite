# Audio-to-Score Experiment Status

This branch contains an unfinished experimental implementation of audio-to-score transcription and voice separation.

The experiment was stopped because real polyphonic audio did not produce sufficiently reliable Sky 15-key arrangements, and the voice-separation search also encountered scalability problems.

This code is preserved for research and historical reference only.

It is not production-ready, is not integrated into the application, and should not be merged into `main` without a new technical evaluation.
