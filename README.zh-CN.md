# SkyMusicPlay Lite 中文版

SkyMusicPlay Lite 是一个早期开发中的 Windows 桌面软件项目。

项目目标是用 Tauri v2、React、TypeScript 和 Rust，循序渐进地重做一个轻量、可维护、适合学习的《Sky: Children of the Light》光遇弹琴辅助工具。

本项目不会一开始就完整复刻旧项目。当前最重要的目标是：每一步都小、稳、能运行或能明确验收，并且让初学者能看懂。

## 当前状态

当前状态：早期开发。

目前还没有创建应用代码。第 0 阶段只建立项目规则和项目说明，暂时不创建 Tauri、React、TypeScript 或 Rust 项目文件。

## 固定技术栈

本项目固定使用：

- Tauri v2
- React
- TypeScript
- Rust
- Vite
- Git 和 GitHub
- Windows 优先

第一版不会使用：

- Electron
- Python
- Vue
- Svelte
- Next.js
- Flutter
- Qt
- C#
- Java
- 纯 Rust GUI

## 第一版范围

第一版必须保持简单、稳定、容易理解。

第一版计划支持：

- 可调整大小的 Windows 桌面窗口
- 基础 React 界面
- 简单文本曲谱输入
- 简单曲谱解析
- 应用内琴键预览
- 预览播放计时
- 播放、暂停、继续、停止控制
- 日志和错误显示
- 最小前端调用 Rust 命令
- Rust 干运行播放流程
- 简单按键映射
- 需要用户手动开启的实验性键盘模拟
- 本地设置保存
- 简单 JSON 曲谱导入
- Windows 构建验证

## 开发原则

本项目遵循以下原则：

- 每次只做当前阶段的内容。
- 每一步都必须可以运行或可以明确验收。
- 不提前实现未来功能。
- 不主动添加复杂依赖。
- 不主动进行大规模重构。
- 每次修改后都解释给初学者听。
- 每个阶段完成后，由用户手动提交和推送 Git。
